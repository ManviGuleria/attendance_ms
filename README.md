📊 Attendance Management System (attendance_ms)
📌 Overview

The Attendance Management System is a software application designed to efficiently manage and track student attendance. It helps automate the process of recording attendance, reducing manual errors and saving time. 
It has a automated warning system that alarms the student when the attendance percentage falls below 75% criteria.


🎯 Features

-> Add and manage student records

->  Mark daily attendance

-> View attendance reports

-> Store and retrieve data efficiently

-> Simple and user-friendly interface

🛠️ Tech Stack

Frontend: HTML , React.js

Backend: Python

Database: MySQL 

Tools: Git, VS Code
